# Join Us 模块说明

## ✅ 已完成功能

### 1. 完整的申请表单页面 (`join-us.html`)

**包含模块：**
- 🎯 职位展示（博士后、博士生、研究助理）
- 📝 在线申请表单
- 📎 CV/简历上传
- 📄 附加文件上传（成绩单、证书等）
- ✨ 现代化UI设计
- 📱 响应式布局

### 2. 表单字段

**个人信息：**
- 姓名（必填）
- 邮箱（必填）
- 电话
- 当前机构

**申请信息：**
- 申请职位（必填）
- 期望入职时间

**教育背景：**
- 最高学历（必填）
- 专业方向（必填）

**研究信息：**
- 研究兴趣（必填）
- 研究经历
- 技术技能

**文件上传：**
- CV/简历（必填，支持 PDF/DOC/DOCX）
- 附加文件（可选，支持多文件）

**其他：**
- 附加信息

### 3. 交互功能

✅ **表单验证**
- 必填字段检查
- 邮箱格式验证
- 文件类型验证

✅ **文件上传**
- 点击上传
- 拖拽上传
- 文件预览
- 大小限制（10MB）

✅ **用户体验**
- 加载状态提示
- 成功提交反馈
- 错误处理
- 平滑滚动

## 🚀 后端集成

要让表单真正工作，需要配置后端。请查看：

📖 **详细指南：** `JOIN_US_BACKEND_GUIDE.md`

**快速方案：**

### 选项1：Formspree（最简单）
1. 注册 https://formspree.io
2. 创建表单获取ID
3. 修改一行代码即可

### 选项2：Netlify Forms
1. 部署到 Netlify
2. 添加 `data-netlify="true"`
3. 自动处理表单

### 选项3：自建后端
1. 使用 Node.js + Express
2. 配置邮件发送
3. 处理文件存储

## 🔒 安全提示

在生产环境中，建议：
1. 添加 reCAPTCHA 防止垃圾邮件
2. 使用 HTTPS 加密传输
3. 验证文件类型和大小
4. 限制提交频率
5. 扫描上传文件的病毒
